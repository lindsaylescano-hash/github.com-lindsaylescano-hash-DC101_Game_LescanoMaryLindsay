# Memory Match Game

## Description
A fun, polished browser-based Memory Match game where players flip cards to find matching pairs. Built as part of the DC 101 Web Development course project. Features scoring, a timer, high score tracking, responsive design, and engaging animations.

## Technologies Used
- HTML5 (semantic structure with ARIA labels)
- CSS3 (advanced styling, animations, responsiveness)
- JavaScript (game logic, DOM manipulation)
- Optional: Web Audio API for sound effects

## Instructions to Run the Game
1. Clone or download this repository.
2. Open `index.html` in any modern web browser.
3. Click "Start Game" to begin â€“ flip cards to match pairs.
4. Aim for the highest score with the fastest time!
5. Use "Reset Game" to restart mid-game.

## Features
- 4x4 grid (16 cards) with dynamic sizing for mobile.
- Score system with penalties; persistent high score.
- Start/Reset buttons for control.
- Smooth animations, sounds, and confetti on win.
- Fully responsive and accessible.
